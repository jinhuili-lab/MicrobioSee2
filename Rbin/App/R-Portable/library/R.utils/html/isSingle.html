Description

Identifies all entries that exists exactly once.

Usage

 isSingle(x, ...)
 singles(x, ...)

Arguments

x

A vector of length K.

...

Additional arguments passed to isReplicated().

Value

A logical vector of length K, indicating whether the value is unique or not.

find_inversions {modelbased}R Documentation

Find points of inversion

Description

Find points of inversion of a curve.

Usage

find_inversions(x)

Arguments

x

A numeric vector.

Value

Vector of inversion points.

Examples

x <- sin(seq(0, 4 * pi, length.out = 100))
plot(x, type = "b")
find_inversions(x)

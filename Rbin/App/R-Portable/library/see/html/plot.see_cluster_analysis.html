plot.see_cluster_analysis {see}R Documentation

Plot method for computing cluster analysis

Description

The plot() method for the parameters::cluster_analysis() function.

Usage

## S3 method for class 'see_cluster_analysis'
plot(x, data = NULL, n_columns = NULL, size_bar = 0.6, ...)

Arguments

x

An object.

data

The original data used to create this object. Can be a statistical model.

n_columns

For models with multiple components (like fixed and random, count and zero-inflated), defines the number of columns for the panel-layout. If NULL, a single, integrated plot is shown.

size_bar

Size of bar geoms.

...

Arguments passed to or from other methods.

Value

A ggplot2-object.

Examples

library(parameters)
groups <- cluster_analysis(iris[, 1:4], 3)
plot(groups)

clearValues {raster}R Documentation

Clear values

Description

Clear cell values of a Raster* object from memory

Usage

clearValues(x)

Arguments

x

Raster* object

Value

a Raster* object

See Also

values, replacement

Examples

 
r <- raster(ncol=10, nrow=10)
values(r) <- 1:ncell(r)
r <- clearValues(r)

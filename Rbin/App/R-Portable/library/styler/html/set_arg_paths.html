set_arg_paths {styler}R Documentation

Standardize paths in root

Description

Standardization required to use setdiff() with paths.

Usage

set_arg_paths(path)

Arguments

path

A path.

See Also

dir_without_.

Examples

styler:::set_arg_paths(c("./file.R", "file.R", "../another-file.R"))

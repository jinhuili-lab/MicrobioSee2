fill_desc {golem}R Documentation

Description

Fill your description

Usage

fill_desc(
  pkg_name,
  pkg_title,
  pkg_description,
  author_first_name,
  author_last_name,
  author_email,
  author_orcid = NULL,
  repo_url = NULL,
  pkg_version = "0.0.0.9000",
  pkg = get_golem_wd()
)

Arguments

pkg_name

The name of the package

pkg_title

The title of the package

pkg_description

Description of the package

author_first_name

First Name of the author

author_last_name

Last Name of the author

author_email

Email of the author

author_orcid

ORCID of the author

repo_url

URL (if needed)

pkg_version

The version of the package. Default is 0.0.0.9000

pkg

Path to look for the DESCRIPTION. Default is get_golem_wd().

Value

The desc object, invisibly.
